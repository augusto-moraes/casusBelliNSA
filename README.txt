Le jeu se lance avec le fichier start (start.bat sous windows et start.sh sous mac et linux),
ou en exécutant le ficher Execute qui se trouve dans le fichier src.
Le code est constitué de 24 classes.


Paysan (soldat niveau 1), Lancier (soldat niveau 2), Chevalier (soldat niveau 3), et Paladin(soldat niveau 4)
héritent toutes les 4 de Soldat, qui elle-même hérite de Unite

Tour(bâtiment défensif le plus faible), Tourlvl2 (bâtiment défensif le plus fort), Mine
(apporte de l'argent et une très petite protection) ,et TownHall (bâtiment qui représente la base du joueur)
héritent toutes les 4 de Batiment, qui elle-même hérite de Unite
