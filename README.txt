Le jeu se lance avec le fichier start (start.bat sous windows et start.sh sous mac et linux),
ou en exécutant le ficher Execute qui se trouve dans le fichier src.
Pour jouer avec le nombre de joueurs souhaités, cliquez sur le bouton correspondant.
Le bouton Start lance le jeu par défaut avec 2 joueurs.

Lien de la vidéo explicative servant de tutoriel et de démonstration (visionnage de la permière moitié conseillée)
https://filesender.renater.fr/?s=download&token=f8812878-7110-4061-a257-dd85ee7b813f

Pour placer une unité, cliquez sur son bouton correspondant, puis cliquez sur la case où vous souhaitez la mettre
pour la placer. Le bouton V vert finit votre tour et passe la main au joueur suivant.

Le Bouton bleu en forme de cercle avec une flèche au bout annule l'action que vous venez de faire
Vous pouvez voir l'argent que vous possédez et ce que vous récoltez (ou perdez) à la fin du tour, sur le côté gauche
de la fenêtre.

Le jeu s'arrête quand l'un des joueurs a tué tous les autres joueurs (joueur mort = joueur n'ayant plus de cases de sa
couleur).
